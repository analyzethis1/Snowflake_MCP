üîß Troubleshooting

‚ö° Quick Fixes

Issue: Application won't start

Solutions:
  • Ensure Python 3.8+ is installed
  • Try: pip install -r requirements.txt
  • Check port 8501 is available
  • Run: streamlit run streamlit_app.py --server.port 8502
Issue: Connection failed (Production Mode)

Solutions:
  • Verify config.json credentials
  • Test with Snowflake's SnowSQL first
  • Check network connectivity
  • Use "Test Connection" button in app
Issue: AI features not working

Solutions:
  • Demo mode has mock AI responses (this is normal!)
  • For real AI: Add Google AI API key to config
  • Run "AI Health Check" in Admin Tools
  • Check debug information in sidebar

üîç Debug Mode
